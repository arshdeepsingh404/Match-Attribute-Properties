# Privacy Policy for MatchAttProp

This Privacy Policy section describes the data practices for the Custom Object Snap application. To promote transparency and protect user privacy, this policy explicitly addresses the mandatory requirements of the Autodesk App Store.

### 1. Data Collection and Usage
This application does not collect, store, or transmit any personal information, user data, or drawing metadata. It functions by reading and modifying the OSMODE system variable within AutoCAD. The 9 user-defined memory slot configurations are saved locally on your computer in the %AppData%\CustomOsnap\ directory. This data is kept completely local and is never transmitted to external servers or local logs.M

### 2. Third-Party Sharing
Custom Object Snap does not share any information with third parties. There are no integrated analytics tools, advertising SDKs, or external tracking mechanisms included in the software.

### 3. Data Retention and Deletion
Because the application does not collect or transmit data to external servers, there is no remote data retention. Local configuration data can be deleted at any time by removing the CustomOsnap.txt file from the AppData directory.

### 4. User Consent and Data Revocation
As no personal data is collected or stored, there is no information for users to request the deletion of. Users can effectively revoke consent at any time by uninstalling the application.
